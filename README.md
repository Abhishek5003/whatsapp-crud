# 💬 WhatsApp CRUD Clone

A beginner backend project inspired by WhatsApp, built using **Node.js**, **Express.js**, **MongoDB**, **Mongoose**, and **EJS**. This project demonstrates complete CRUD (Create, Read, Update, Delete) operations using MongoDB.

> **Note:** This project was created for learning backend development and understanding how Express.js interacts with MongoDB using Mongoose.

---

## 🚀 Features

- 💬 Create a new chat
- 📋 View all chats
- ✏️ Edit existing messages
- 🗑️ Delete chats
- 📦 Store data in MongoDB
- 🌐 Server-side rendering using EJS
- 🔄 RESTful routing with Express

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/Abhishek5003/whatsapp-crud.git
```

Install dependencies

```bash
npm install
```

Start MongoDB locally

Run the application

```bash
node index.js
```

Open your browser

```
http://localhost:8080
```
